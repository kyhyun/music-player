# React Music Player

## 프로젝트 개요
음악 목록들을 실행할 수 있는 뮤직 플레이어 앱

## 🎄 프로젝트 구조

```
src
 ┣ assets       // images, music, svg 파일
 ┣ data         // json 파일
 ┣ hooks        // Custom Hooks
 ┣ store        // 전역 state
 ┣ pages        // 페이지
 ┣ styles       // 전역 style
 ┣ types        // 필요한 type 정의
```

##  실행 방법

1. Repository 클론
```sh
$ git clone https://github.com/kyhyun/music-player.git
```

2. Dependencies 설치
```sh
$ yarn install
```

3. Run 실행
```sh
$ yarn start
```
